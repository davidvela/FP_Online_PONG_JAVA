if the app is not possible to run making double click; you have to run it from the terminal.

Windows: 	Start, programs, accesories, command_prompt.exe
		if the app is in the file Downloads: write in the terminal:
			cd Downloads
			java -jar ClientLab.jar

Linux/Mac	terminal
		if the app is in the Desktop write in the terminal: 
			cd Desktop
			java -jar ClientLab.jar


Usually this is mandatory in Linux and Mac.

Furthermore, if there is another mistake, write in the terminal:
	java -version

If the version of java is lower than 1.6, try to update the java version.
